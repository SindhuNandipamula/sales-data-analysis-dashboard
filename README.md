# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview
This project analyzes sales data using Excel and presents insights through an interactive dashboard.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## 📊 Features
- Sales analysis by Region
- Sales analysis by Category
- Monthly sales trend
- Interactive filters using slicers
- KPI metrics (Total Sales, Orders, Average Sales)

## 💡 Key Insights
- West region generates highest sales
- Technology category performs best
- Sales show seasonal trends across months
