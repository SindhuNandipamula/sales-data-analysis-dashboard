# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview

This project analyzes sales data and presents insights through interactive dashboards built using **Microsoft Excel and Power BI**. The goal is to understand sales performance and support data-driven decision-making.

## 🛠 Tools Used

* Microsoft Excel
* Power BI
* Pivot Tables
* Data Cleaning (handling missing values)

## 📊 Features

* Sales analysis by Region
* Sales analysis by Category
* Monthly sales trend (time-series analysis)
* KPI metrics (Total Sales, Total Orders, Average Sales)
* Interactive filtering:

  * Excel (Slicers)
  * Power BI (Slicers)

## 💡 Key Insights

* The West region generates the highest sales.
* The Technology category contributes the most revenue.
* Sales show variation across months, indicating seasonal trends.

## 📂 Files Included

* `Sales_Dashboard.xlsx` (Excel Dashboard)
* `Sales_Dashboard.pbix` (Power BI Interactive Dashboard)
* Dataset used for analysis
